Lapis 1.16.0 Reference Manual

Welcome to the Lapis reference manual. You can find a list of all the guides to the left. Because Lapis has can be used in both Lua and MoonScript, but the conventions or syntax may vary, you can toggle which language all examples are in with the buttons on the upper left.

you can toggle language of the examples in the documentation from the sidebar. Some guides only apply to a certain language.

If you're not sure what Lapis is you can find a general overview on the homepage: https://leafo.net/lapis.

Lapis is open source, you can find the souce code here: https://github.com/leafo/lapis.

If you are having any problems with Lapis open an issue on the issues tracker.
